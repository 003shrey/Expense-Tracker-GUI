# Expense Tracker GUI – Python & Tkinter

This is a simple desktop application built with Python and Tkinter that helps you track your day-to-day expenses. The app allows you to record, view, delete, and analyze your spending with ease. All expense data is stored locally in a CSV file.

---

## Features

- Add expenses with amount, category, and description
- View all transactions in a structured table format
- Delete specific entries
- View a summary report with category-wise totals
- Display expenses graphically using a bar chart
- Lightweight and works completely offline

---
